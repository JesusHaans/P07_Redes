para ingresar a la calculadora usaremos el 
comando:
 
``` BASH
$ python3 vlsm.py
```
Con esto entraremos a la version (y unica si no 
nos da tiempo) de standar inpuput y output de la 
calculadora.

agreguen el numero de Host para cada subred de
